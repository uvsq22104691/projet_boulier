# Projet boulier
## Fonctionalités
- Sauvegarder / charger un boulier
- reinitialiser le boulier
- mode simulation
    - bouger les boules en cliquant dessus
- options
    - vitesse des boules
    - nombre de colonnes du boulier
    - clignotement des boules lors du déplacement
- mode opération
    - addition
    - soustraction
    - multiplication
